设计说明

  • 双灵感色（v3 核心变化）：每个方案有两个灵感色。主色（accent）占 60-70%，用于标题、主按钮、导航高亮、装饰线等核心元素；副色（accent2）占 30-40%，用于标签、次要按钮、波点背景等辅助元素
  • 双色波点背景：body::before 使用两个 radial-gradient 交替排列，通过 background-position 偏移半个周期实现棋盘式双色波点。CSS 变量 --dot-color-1 和 --dot-color-2 分别控制两种波点颜色
  • 副色使用场景：标签（tag）使用副色背景 + 副色文字；次要按钮使用副色边框 + 副色文字；波点背景中副色与主色交替出现，形成丰富的视觉层次
  • 阅读优先：所有配色以长时间阅读舒适度为第一目标，背景使用微妙的色调倾向（而非纯白），减少视觉疲劳
  • 卡片半透明：卡片使用 rgba(255,255,255,0.7) 半透明背景 + backdrop-filter: blur(10px) 毛玻璃效果，让背景波点若隐若现
  • 边框透明度：边框使用主题色的低透明度（12%-15%），比纯灰色边框更有设计感，同时不会太抢眼
  • 文字层次：主文字使用带色调倾向的深色（如 #2a3a38 而非纯黑 #000），次要文字和灰色文字层次分明
  • 暗色模式：暗色背景使用灵感色的极深版本（如 #0a1a1a 而非纯黑），卡片使用半透明深色，双色波点在暗色模式下透明度略微提高以保持可见性
  • 直角设计：所有元素保持直角（border-radius: 0），符合网站哥特风格
  • 字体：标题使用 Noto Serif SC 衬线体，正文使用 Noto Sans SC 无衬线体，哥特装饰符号 ✦ 贯穿全站
